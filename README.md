# Portfolio Profissional

Projeto de pagina de apresentacao com visual moderno, foco em posicionamento e conversao profissional.

## Arquivos

- `index.html`: estrutura da pagina
- `styles.css`: visual, layout responsivo e animacoes
- `script.js`: dados do perfil e renderizacao de conteudo

## Personalizacao rapida

Edite o objeto `profileData` em `script.js`:

- nome e cargo
- texto de abertura e sobre voce
- links (LinkedIn, GitHub, email)
- metricas de impacto
- skills
- projetos com resultados

## Como abrir

Abra `index.html` no navegador.

Opcional com servidor local (Node):

```bash
npx serve .
```

Depois acesse a URL informada no terminal.
